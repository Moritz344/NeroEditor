# Features
- Basic Syntax Highlighting for Python
- open text files,python files
- save text file,python files
- change font
- change font size
- change colorscheme
- change dark/light mode
- See Recent Files
- See Current File
- Ctrl+z(undo)
- Ctrl+y(redo)
- file info window
